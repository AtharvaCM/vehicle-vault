# Product Roadmap

## Phase 1: MVP (0 → Launch)

Focus:

- Core functionality
- Fast logging
- Basic reminders

Features:

- Auth
- Vehicle CRUD
- Maintenance logs
- Attachments
- Reminders
- Dashboard
- Export

---

## Phase 2: Usability Improvements

Focus:

- Reduce friction
- Improve retention

Features:

- Quick add service presets
- Pre-filled service templates
- Better dashboard insights
- Odometer history tracking
- Reminder presets (India-specific)

---

## Phase 3: Smart Features

Focus:

- Automation
- Intelligence

Features:

- Invoice OCR parsing
- Auto-suggest service entries
- Smart reminders (usage-based)
- Service schedule templates by vehicle

---

## Phase 4: Ecosystem Expansion

Focus:

- Integrations

Features:

- Registration number auto-fetch
- Insurance integrations
- Service center integrations
- Garage sync

---

## Phase 5: Advanced Product Layer

Focus:

- Power users & scale

Features:

- Multi-user vehicle sharing
- Advanced analytics
- Vehicle health score
- Resale report generator
- API access

---

## Long-Term Vision

Become:
> “The single source of truth for vehicle ownership”
